🧹 Chat History Cleaner

Включить очистку
Логика очистки:
Оставляются первые N и последние N блоков сообщений.
Блоки в середине полностью удаляются из DOM для снижения нагрузки.
После очистки расширение завершает работу.
Используйте, если страница долго загружается и очистка не сработала.
Отладка для поддержки
Откройте нужный чат, нажмите «Собрать данные», затем «Копировать» и вставьте JSON в сообщение.
Настройки сохранены!
