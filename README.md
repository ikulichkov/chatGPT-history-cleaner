# Chat History Cleaner

Chrome расширение для снижения нагрузки на компьютер путем удаления предыдущих блоков сообщений в чате.

## Описание

Расширение автоматически удаляет блоки сообщений в середине истории чата, оставляя только первые N и последние N блоков. Это значительно снижает нагрузку на компьютер, уменьшая размер DOM и количество элементов на странице.

## Функции

- ✅ Удаляет блоки сообщений в середине истории
- ✅ Оставляет первые N и последние N блоков (настраиваемо)
- ✅ Полностью удаляет элементы из DOM (не скрывает)
- ✅ Автоматически завершает работу после очистки
- ✅ Показывает статистику: размер страницы до и после очистки
- ✅ Настраиваемое количество блоков (первые: минимум 0, последние: минимум 1)

## Установка

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку с расширением

## Использование

1. Откройте чат (например, ChatGPT)
2. Нажмите на иконку расширения в панели инструментов
3. Настройте количество видимых блоков или отключите функцию
4. Нажмите "Сохранить настройки"

## Настройки

- **Включить очистку**: Включает/выключает функцию удаления блоков
- **Количество первых блоков**: Количество первых блоков, которые будут оставлены (минимум 0)
- **Количество последних блоков**: Количество последних блоков, которые будут оставлены (минимум 1)

## Технические детали

Расширение использует:
- Manifest V3
- Content Scripts для манипуляции DOM
- Chrome Storage API для сохранения настроек
- MutationObserver для отслеживания новых сообщений
