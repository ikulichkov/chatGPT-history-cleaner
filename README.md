# Chat History Cleaner

Chrome расширение для снижения нагрузки на компьютер путем скрытия предыдущих блоков сообщений в чате.

## Функции

- ✅ Скрывает предыдущие блоки сообщений
- ✅ По умолчанию показывает только последние 10 блоков
- ✅ Настраиваемое количество видимых блоков
- ✅ Возможность включить/отключить функцию
- ✅ Кнопка "Показать ответ" для раскрытия скрытых блоков
- ✅ Автоматическое обновление при появлении новых сообщений

## Установка

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку с расширением

## Использование

1. Откройте чат (например, ChatGPT)
2. Нажмите на иконку расширения в панели инструментов
3. Настройте количество видимых блоков или отключите функцию
4. Нажмите "Сохранить настройки"

## Настройки

- **Включить очистку**: Включает/выключает функцию скрытия блоков
- **Количество видимых блоков**: Количество последних блоков, которые будут видны (от 1 до 100)

## Технические детали

Расширение использует:
- Manifest V3
- Content Scripts для манипуляции DOM
- Chrome Storage API для сохранения настроек
- MutationObserver для отслеживания новых сообщений
